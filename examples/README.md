# PyKarta Examples

## geojson-simple-editor

An editor for GeoJSON files based on the GeoJSON widget example

## pdf\_map.py

Render a map to a PDF file

## wms\_server.py

A simple WMS server with a PyKarta backend

## converters

* geojson2osm -- Convert a GeoJSON file into Openstreetmap format
* osm2geojson -- Convert an Openstreetmap file to GeoJSON format
* osm2osm-county -- Extract county borders from an Openstreetmap file
* shp2gpx -- Convert a Shapefile to GPX format
